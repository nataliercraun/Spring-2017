Detailed Description

template<typename Key, typename Value>
class Node< Key, Value >

A templated class for a Node in a search tree. This represents a node in a normal binary search tree, but can also be extended in the future for other kinds of search trees, such as Red Black trees, Splay trees, and AVL trees. You do NOT need to implement any functionality or add additional data members or helper functions.
